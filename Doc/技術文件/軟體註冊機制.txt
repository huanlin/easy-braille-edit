易點雙視 4.x 軟體註冊機制

應用程式啟動時，去 github 讀取 users.lic 檔案，並將此檔案編碼之後，保存在應用程式所在目錄。
如果沒有網路，或者無法下載 users.lic，則檢查本機是否有這個檔案。
  有：檢查 windows registry 是否有序號，若有，再跟 users.lic 比對是否為合法序號。
      序號合法：正常使用。
	  序號無效：提示使用者序號無效，將無法列印和輸出文件。
	  呼叫 heroku 網站的 api 把註冊
  無：開啟註冊視窗。輸入序號和使用者資料，並且跟 users.lic 比對序號是否正確。
      註冊成功：把序號寫入 windows registry。
	  註冊失敗：
	  取消註冊：提示無法列印和輸出雙視文件。


GitHub 的 users.lic 檔案

URL: 參見 EasyBrailleEdit.Common.Constant.DefaultAutoUpdateRootUrl 字串常數。

檔案保存於使用者機器端時，會先以單純的 Base64 編碼之後才存至檔案。

未編碼的檔案格式與範例：

序號--------------- 有效期限-- 狀態 客戶名稱
1111-2222-3333-4444 2018-02-20 0    台北市視障者家長協會
1111-2222-3333-4444 2018-02-20 1    台北市視障者家長協會

｢狀態｣欄用來代表該序號的狀態：
0 表示正常可用
1 表示禁止（例如研判該序號可能已經被盜用）
